## What is Kivai?

Kivai is a platform interoperability standard that enables any AI system to safely and deterministically control physical devices using a standardized intent contract (schema) and device-facing APIs.

Kivai is not an AI assistant.
Kivai is the execution and interoperability layer between AI systems and device ecosystems.
