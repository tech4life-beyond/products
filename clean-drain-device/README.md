## Overview

The Clean Drain Device (also referred to as the DrainClean T Adapter) is an inline HVAC condensate drain solution designed to prevent overflow damage, enable early clog detection, and simplify maintenance.

This repository contains the complete official TOIL Product Pack documentation required for licensing, validation, and commercial deployment.
